What is TurboPutative?

TurboPutative is a program made up of a set of tools that automate and streamline the processing of tables with putative annotations, that is, of tables that in each row assign a candidate metabolite to the signals detected by the mass spectrometer (feature). The function of each tool is explained in Modules section.
